## Regras de Atualização

1. **Sempre use versões exatas** para TypeScript (sem ^ ou ~)
2. **Use `workspace:*`** para dependências internas do monorepo
3. **Mantenha React e Next.js sincronizados** em todos os workspaces
4. **Atualize dependências UI** primeiro no packages/ui, depois nas apps
5. **Teste cada atualização** rodando `pnpm check-types` e `pnpm lint`

## Comandos Úteis

```bash
# Verificar versões desatualizadas
pnpm outdated -r

# Atualizar dependências interativamente
pnpm update -i -r --latest

# Verificar duplicações
pnpm dedupe

# Limpar e reinstalar
pnpm clean && pnpm install
```
